# Project Setup Instructions

Use this guide to set up the project from scratch. **You usually don't need
this — the F8 template ships with all of this already wired up.** These docs
exist so you can understand how each piece is built, rebuild from scratch,
or swap out any layer.

Write the complete code for every step. Do not get lazy. Write everything
that is needed.

Your goal is to completely finish the project setup.

## Run in this order

Follow these guides in order — each one assumes the previous step is done:

1. [`setup-frontend.md`](./setup-frontend.md) — Next.js 14, Tailwind, shadcn primitives.
2. [`setup-backend.md`](./setup-backend.md) — Supabase Postgres, Drizzle ORM, Server Actions.
3. [`setup-auth.md`](./setup-auth.md) — Supabase Auth (email/password + OAuth), middleware, protected routes.
4. [`setup-payments.md`](./setup-payments.md) — Stripe Checkout, Billing Portal, webhook.

For adding a new database table later, see
[`setup-tables.md`](./setup-tables.md).

- The project is now setup.
